# project-management

An agent skill for managing projects using a structured planning and execution system.

## Install

```sh
npx skills add /path/to/project-management
```

## Usage

Trigger this skill by mentioning "project" or "projects" in any context:

- **Discovery**: "what projects are pending?", "show me the backlog"
- **Planning**: "create a project for X", "add an issue to Y"
- **Execution**: "work on the next issue", "continue the backup-automation project"
- **Organization**: "archive completed projects", "move X to backlog"

## Diagrams

Project plans embed mermaid architecture diagrams, compatible with editors like VS Code with Markdown Preview Enhanced.
